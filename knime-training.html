KNIME Training

KNIME Analytics Platform is a leading open source option for data-driven innovation, built on a modular data pipelining concept, with powerful machine learning and data mining capabilities.

Learn how to install and use KNIME to discover the potentials hidden in your data, mine for fresh insights, or implement advanced predictive and prescriptive analytic models, without any programming.

Why Learn KNIME

  • KNIME makes Understanding data, and designing and documenting data science workflows and reusable components accessible to everyone
  • KNIME is the perfect data science tool for deploying across your entire organisation. Whether you are wrangling Excel, building ETL workflows or developing Machine Learning models. Start for FREE and expand as your data needs grow
  • No coding experience is required, yet KNIME still allows you to use languages like R, Python or Java if you want. So, you can code if you want but don't have to!

Duration: 3 days

Target Audience:

  • Anyone searching for user-friendly, easily understandable, codeless and highly useful tool for data analysis and machine learning tasks without the necessity of having programming skills
  • Excel users who want to learn better ways to prepare data. Excel is nice but KNIME is AWESOME!
  • Data scientists / analysts who want to work smarter, faster and more efficiently

Learning Outcome

By the end of the training, participants will learn how to:

  • Install and configure KNIME
  • Organize, clean, and transform data sets for analysis
  • Build Data Science scenarios
  • Train, test and validate machine learning models

Course Format:

  • Interactive lecture and discussion
  • Lots of exercises and practice
  • Hands-on implementation in a live-lab environment
  • Introduction to KNIME:
    • Overview of Data Analytics
    • What is KNIME? Who uses it? What can we do with it?
    • Installation and customization
  • Exploring the KNIME Workbench:
    • Nodes and workflows
    • The node repository
    • Node creation and basic commands
    • Annotation and comments
    • Search and Help
  • Accessing Data:
    • The Excel Reader Node, File Reader Node, Table Reader Node, etc.
  • Cleaning Data:
    • Describe data and verify data quality
    • Explore data with Scatterplot and Boxplot
    • Row & Column filtering, String manipulation, Missing value
    • Column Rename and Resorter
    • Writing to a file: CSV Writer
  • Transforming Data:
    • Modeling data
    • Combining tables
    • Aggregating values with GroupBy and Pivoting
  • Lab Work 1: Sales Report Automation
  • Machine Learning with KNIME:
    • The business value of learning machines
    • Three types of learning algorithms:
      - Supervised learning
      - Unsupervised learning
      - Reinforcement learning
      - Selecting the right learning algorithm
    • Evaluating performance:
      - Regression
      - Classification
      - Underfitting and overfitting
      - Validating a model
    • Overview of Predictive modeling process flow
  • Lab Work 2: Predicting numbers through regressions
  • Lab Work 3: Segmenting customers with clustering

Participants will need their own Laptop to download and install KNIME Analytics Platform. A basic understanding of making sense of the data is expected
